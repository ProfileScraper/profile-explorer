# Profile Explorer

Community-contributed scraping profiles for [ProfileScraper](https://github.com/ProfileScraper/configurable-scraper). Pre-configured field selectors, pagination, and anti-detection settings for popular e-commerce, real estate, and other websites. Clone ready-to-use profiles or contribute your own.

## Using Public Profiles

Public profiles are automatically available in ProfileScraper:

1. Open ProfileScraper desktop app
2. Navigate to **Profile Explorer** tab
3. Click **Refresh** to sync latest profiles
4. Browse, search, and filter profiles by tags
5. Click **Clone** to create an editable copy

## Contributing Profiles

### Via ProfileScraper App (Easiest)

1. Create and test a profile in ProfileScraper
2. Click **"Publish to Profile Explorer"** button
3. Login with GitHub
4. Add description and tags
5. Click **"Create Pull Request"**
6. Your profile will be reviewed and merged

### Manually

See [CONTRIBUTING.md](CONTRIBUTING.md) for manual submission guidelines.

## Available Profiles

Profiles are stored in `public-profiles.json`. Browse them in the ProfileScraper app or view the JSON file directly.
